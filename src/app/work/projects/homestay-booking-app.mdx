## Overview
Homestay Booking App is a Flutter application that allows users to discover and book homestay accommodations. Users can register, log in, and manage their bookings, while administrators can view and manage user registrations and bookings.

## Features
- User registration and authentication
- User profile management
- Booking form submission (multiple homestays)
- View and update booking details
- Administrator panel for user and booking management
