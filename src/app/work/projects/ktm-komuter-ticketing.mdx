## Overview

The **KTM Komuter Online Ticketing System** is a comprehensive web-based platform that modernizes the ticket purchasing experience for KTM Komuter passengers. Built using **ASP.NET MVC with C#**, this system provides an intuitive interface for passengers to purchase tickets while offering special discounts for senior citizens, disabled individuals, and students.

The platform features automated email notifications, QR code generation for tickets, and comprehensive reporting tools for management to analyze travel patterns and transaction data.
