## Overview
SQCM Catering Booking System is a web application that allows users to book catering services online. The system is built with PHP and uses Docker for containerized local development and deployment. Admins can manage bookings, menus, and customer information efficiently.

## Features
- User registration and login
- Online catering booking form
- Menu management for admins
- Booking management for admins
- Docker integration for easy setup and deployment
- MySQL database for data storage
